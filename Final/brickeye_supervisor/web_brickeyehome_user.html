人員管理系統區塊

功能: 可以新增、刪除、修改使用者名稱和密碼 。如重複名稱的帳號將無法進行新增。

搜尋區塊

功能: 依據時間區間顯示該時間段的檢測品詳細數據，並可透過選取特定 ID 檢測單一品項進行隨機抽樣。

同時顯示當前選取時間段的:

  • 產品總數：
  • 良品總數：
  • 劣質品總數：
  • 劣質品比率：

以確認總體是否產生過多缺陷品

品質篩檢區塊

功能: 依據選取特定條件顯示出包含符合條件的產品 (例如品質為良品就只會顯示良品)，並設置限定資料顯示功能(初始設置50筆)，避免過量資料載入的情況。可透過調整起始、結束資料數字顯示特定段列的資訊。

儀錶板區塊

功能: 確認缺陷時間序列 (Time Series) 圖表，以下包含:

  • 當前時間一小時內的缺陷數量，並展現出最高峰缺陷數量的三個時段以利檢查
  • 當前時間六小時內缺陷嚴重超標 (以每五分鐘為一個區間)，採用Grafana即時監測警告，並加入警告失靈時的預備措施(超標可視化，這裡設置為>=10)
  • 當前時間二十四小時內缺陷數量，可以確認當日整體的缺陷品產出頻率，並依據需求縮放進行確認
  • 依據時間區間獲取缺陷數量趨勢圖，如第三張圖表的功能，但是可以透過選取時間區間檢查特定的日期

數據模型區塊

功能: 展現訓練數據的模型折線圖，並比對現在和過去的缺陷檢測模型，確認哪個為較優的選項
